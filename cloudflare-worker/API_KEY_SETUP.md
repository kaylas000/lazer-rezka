# Установка Groq API ключа

## Шаг 1: Получите ключ

1. Зарегистрируйтесь на https://console.groq.com
2. Перейдите в **API Keys**
3. Нажмите **Create API Key**
4. Скопируйте ключ (начинается с `gsk_...`)

## Шаг 2: Вставьте ключ в код

Откройте файл `cloudflare-worker/worker.js` и найдите строку:

```javascript
const GROQ_API_KEY = 'ВСТАВЬТЕ_ВАШ_КЛЮЧ_СЮДА';
```

Замените на ваш реальный ключ:

```javascript
const GROQ_API_KEY = 'gsk_abc123xyz456...';
```

**ВАЖНО:** Этот файл будет в приватном репозитории GitHub, поэтому ключ защищён от публичного доступа.

## Шаг 3: Деплой на Cloudflare

1. Откройте https://dash.cloudflare.com
2. **Workers & Pages** → **Create application** → **Create Worker**
3. Скопируйте весь код из `worker.js` в редактор Cloudflare
4. **Save and Deploy**

Готово! AI-консультант работает.

## Безопасность

✅ Репозиторий GitHub приватный — никто не видит ключ
✅ Cloudflare Worker изолирован — ключ не виден на сайте
✅ Rate limiting защищает от злоупотреблений

## Проверка работы

После деплоя откройте ваш сайт и нажмите на кнопку чата в правом нижнем углу.
