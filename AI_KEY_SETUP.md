# Установка AI-консультанта за 2 шага

## Шаг 1: Получите ключ Groq (бесплатно)

1. Зайдите на https://console.groq.com
2. Зарегистрируйтесь
3. **API Keys** → **Create API Key**
4. Скопируйте ключ (начинается с `gsk_...`)

## Шаг 2: Создайте файл с ключом

В папке `assets/js/` создайте файл `api-key.js` со следующим содержимым:

```javascript
// Конфигурация API ключа

const API_CONFIG = {
  GROQ_API_KEY: 'ваш_ключ_groq_сюда'
};
```

Замените `'ваш_ключ_groq_сюда'` на ваш реальный ключ.

## Готово! 🎉

AI-консультант работает на сайте.

---

## Безопасность

✅ **Файл `api-key.js` скрыт от GitHub** (в `.gitignore`)  
✅ **Файл `api-key.js` скрыт от поисковиков** (в `robots.txt`)  
✅ **Шаблон `api-key.template.js` в Git** (без реального ключа)

Ключ виден только тем, кто откроет исходный код сайта в браузере. Для полной защиты используйте Cloudflare Worker (см. `cloudflare-worker/API_KEY_SETUP.md`).

---

## Если файл не создаётся

Скопируйте шаблон:
```bash
cp assets/js/api-key.template.js assets/js/api-key.js
```

Затем отредактируйте `api-key.js` и вставьте ваш ключ.
