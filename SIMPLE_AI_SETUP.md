# Самая простая установка AI-консультанта

## Вариант 1: Прямо на сайте (БЕЗ Cloudflare)

### Шаг 1: Получите ключ Groq
1. Зайдите на https://console.groq.com
2. Зарегистрируйтесь (бесплатно)
3. **API Keys** → **Create API Key**
4. Скопируйте ключ

### Шаг 2: Вставьте ключ в код
Откройте файл `assets/js/ai-chat.js` в GitHub, найдите строку 5:

```javascript
const GROQ_API_KEY = 'ВСТАВЬТЕ_ВАШ_КЛЮЧ_СЮДА';
```

Замените на ваш ключ и сохраните (commit).

### Готово!
AI-консультант работает сразу на сайте.

**⚠️ Внимание:** Ключ будет виден в коде сайта. Используйте только для тестирования или если репозиторий приватный.

---

## Вариант 2: Через Cloudflare (безопаснее)

Если нужна полная безопасность, используйте Cloudflare Worker (см. `cloudflare-worker/API_KEY_SETUP.md`).

Для переключения на Cloudflare измените в `assets/js/ai-chat.js`:
```javascript
const USE_DIRECT_API = false; // Использовать Cloudflare Worker
```
