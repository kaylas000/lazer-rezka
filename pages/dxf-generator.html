Генератор DXF-файлов

Настройте параметры детали, скачайте готовый DXF-файл и узнайте примерную стоимость

📘 Как пользоваться генератором — пошаговая инструкция

1. Выберите тип детали — пластина, фланец/диск или кронштейн. Для каждого типа своя форма слева.

2. Задайте размеры — ширина, высота, радиус углов. Для фланца — наружный и внутренний диаметр. Для кронштейна — выберите тип (L/П/Z) и размеры.

3. Расставьте отверстия — кликните мышью по предпросмотру справа где нужно отверстие. Каждое отверстие появится в таблице под формой. Там можно точно задать координаты X, Y и диаметр каждого отверстия.

4. Чтобы удалить отверстие — кликните по нему на предпросмотре или нажмите × в таблице.

5. Выберите материал, толщину и количество — цена рассчитается автоматически по длине реза.

6. Нажмите «Скачать DXF» — файл сохранится на компьютер, готовый для загрузки в станок.

⚠️ Цена примерная — не учитывает точку входа лазера. Точная стоимость после проверки чертежа технологом.
⚠️ Минимальный диаметр отверстия не может быть меньше толщины металла.
⚠️ Центр координат (0,0) — центр детали. Положительные X — вправо, положительные Y — вверх.
